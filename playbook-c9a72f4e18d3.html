The Campaign

Families of these students paused for summer or paused with open-ended language during the summer window. We're sending them the Fall is Calling email offering a GameU merch code when they get back on the fall calendar. Deadline for families to book: Friday, September 11.

Coordinator Workflow — when a reply comes in

  1. Confirm identity — match the reply to a student in this list. Verify parent email matches.
  2. Pull their record — open their Pipedrive deal (link in the table). Note what happened when they paused (see "Break context" column).
  3. Find a fall slot — use the Scheduling Bot or Instructor Lookup to find an instructor who matches their subject / availability. If they had a previous instructor still on staff, offer that first.
  4. Book the first class in XP — confirm date, time, instructor with the family.
  5. Move the Pipedrive stage — from Lost to Attending (or Scheduled → Attending once they've had their first class).
  6. Send the code — reply to the parent with their unique code + link to spcustomgear.store. Copy Erin so we track which codes went out.
  7. Log the outcome — add a Pipedrive activity noting the return date, and tag the deal summer-return-2026 so we can measure the campaign later.
